# Mini-Tender-Management-System
A core Java project that simulates a tender management system with bidding and lowest bidder selection.

## Description
This is a console-based Java application that simulates a tender management system. It allows an admin to create tenders and vendors to place bids. The system automatically selects the lowest bidder.

## Features
- Add and view tenders
- Place and view bids
- Automatic winner selection
- File-based data storage

## How to Run

1. Go to src folder:
   cd src

2. Compile:
   javac *.java

3. Run:
   java Main

## Project Structure

- src/ → source code
- test-data/ → sample data
- screenshots/ → output images
  
## Technologies Used
- Core Java
- OOP Concepts
- File Handling
